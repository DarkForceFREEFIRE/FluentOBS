Theme Installation Guide

Follow these simple steps to install and enable the Fluent (Dark) theme in OBS Studio.

1

Prepare the Files

Download the release package from the repository and extract the contents. Ensure you have the following file structure ready before moving them:

Your Extracted Folder
fluentdark.obt (Theme configuration)
fluentdark.qss (Styles and layouts)
fluentdark (Custom SVG assets folder)
Important Folder Name Rule: The assets folder must be named exactly fluentdark (all lowercase, no spaces). This ensures OBS Studio's internal resource paths can locate and load the custom icons.
2

Move to Custom OBS Themes Directory

Copy the configuration file, style sheet, and folder into your operating system's custom OBS themes directory:

Press Win + R, paste the path below, and hit Enter. Move the files there:

%appdata%\obs-studio\themes

Alternative system-wide path

%programfiles%\obs-studio\data\obs-studio\themes\

Open Finder, press Cmd + Shift + G, paste the path below, and move your files there:

~/Library/Application Support/obs-studio/themes/
Flatpak Installation:
~/.var/app/com.obsproject.Studio/config/obs-studio/themes/
Standard/Ubuntu/Native Installation:
~/.config/obs-studio/themes/
3

Enable the Theme inside OBS

Apply the layout inside your OBS Settings panel:

  • Open OBS Studio on your desktop.
  • Go to Settings > General.
  • Locate the Theme dropdown setting at the top.
  • Choose Fluent (Dark) from the list.
  • Click Apply, then click OK to save.
